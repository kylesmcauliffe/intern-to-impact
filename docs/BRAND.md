# Intern to Impact — brand cheat sheet

## Name

**Intern to Impact** — career guides for young professionals (especially first-gen, Bay Area, state-school paths into corporate and tech).

## Voice

- Practical, honest, first-person when Ari writes
- First-gen friendly — no assumed family playbook
- Professional but not corporate-stiff; demure, polished, not performative

## Colors

| Token | Hex | Use |
|-------|-----|-----|
| White | `#FFFFFF` | Page sections |
| Off-white | `#FAFAF8` | Shell background |
| Text | `#1A1A1A` | Headlines and body |
| Muted | `#6B6B6B` | Secondary text |
| Border | `#E8E8E8` | Dividers |

**Pastel accents** (tags and cards only):

| Name | Hex |
|------|-----|
| Lavender | `#E8E0F4` |
| Blush | `#F5E6E8` |
| Mint | `#E0F4EC` |
| Butter | `#F5F0E0` |

Sephora reference: **black & white chrome**, colorful **content blocks** — not a store layout.

## Typography

- **Geist** (Google Fonts) — weights 400, 500, 600
- No serif for v1

## Logo

- **Favicon:** app icon in `public/`
- **Wordmark:** Intern → Impact (black text, arrow between)

## Content tags

| Tag | Type |
|-----|------|
| `guide` | Step-by-step how-tos |
| `story` | Essays, story time |
| `template` | Copy-paste resources |

## Site structure

- Home → Start here + latest posts
- Blog → all posts + tag filters
- About → Ari’s story
- Contact → reach out
